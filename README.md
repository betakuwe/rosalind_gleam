# rosalind_gleam

Just doing some ROSALIND problems in Gleam!

Each problem is located in `src/<problem title>/`

## Running solutions

In the project root, run `gleam run <problem>`.

Example:

```
$ gleam run dna 
  Compiled in 0.01s
  Running rosalind_gleam.main
Solution to dna:
222 209 218 208
  
```
